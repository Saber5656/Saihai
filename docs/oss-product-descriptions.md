## 2. Dev CLI 星系(12)

| P27 | seek | ファイル名・全文・意味検索のハイブリッドでローカルファイルを探す常駐検索(fzf 連携)。 |
| P28 | shellcheckup | シェル起動の遅さをプロファイルし、犯人(重い plugin・PATH 重複)の除去を提案する。 |
| P29 | dnslens | 「なんで繋がらない?」を DNS・証明書・経路・プロキシまで一発診断し、平文で説明する。 |
| P30 | envdiff | 「昨日は動いたのに」を環境スナップショット(env・依存・ツール ver)の比較で特定する。 |
| P31 | worklog | shell 履歴・git・エディタ操作から日報・standup 報告を自動生成する。 |
| P32 | culprit | 「どのコミットで壊れた?」を全自動化する(bisect+環境再現+原因コミット報告)。 |
