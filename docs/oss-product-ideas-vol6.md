# OSS プロダクト提案書 vol.6 — CLI × AI × ゲーム × 遊び心 × MCP の 150 案

**フィードバックの反映:** vol.4〜5 は家庭・計測系に寄りすぎた。発案者の関心軸は
**CLI / AI / ゲーム / 遊び心 / MCP**(開発者としての「自分が欲しい」)であり、
目標は一貫して**「OSS 界隈が盛り上がって成長するプロダクト」**。

vol.6 の選定条件は 3 つの積:
1. **上記 5 軸の交点にある**(全案が最低 2 軸を跨ぐ)
2. **vol.5 基準を維持**: プロンプトラッパー禁止。決定論の技術コアを持つ
3. **コミュニティが自走する構造**(プラグイン・お題投稿・対戦・コレクション・日替わり)

既出(採用済み・不採用)とは重複しない。名称仮・競合メモは 2026-07 知識ベース。
