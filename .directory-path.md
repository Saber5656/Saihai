# Directory Path Aliases

This tracked file is a public alias catalog. Do not put concrete machine paths,
usernames, private vault names, or environment-specific examples here.

Set actual local values in ignored `.directory-path.local.md` files or matching
environment variables.

| Alias | Local value | Purpose |
|---|---|---|
| `SAIHAI_ROOT` | set locally | Saihai source repository |
| `AGENT_TEAMS_VIEWER_ROOT` | set locally | Saihai source repository (legacy Agent-Teams-Viewer alias, kept for compatibility) |
| `AGENTS_VAULT_ROOT` | set locally | shared organization vault |
| `USER_VAULT_ROOT` | set locally | personal Obsidian vault |
| `SKILLS_REPO_ROOT` | set locally | skills source repository |
| `SKILLS_ROOT` | set locally | skill definitions |
| `DOTFILES_ROOT` | set locally | local agent/runtime configuration |
| `DEV_ROOT` | set locally | local source repositories |
| `DEV_WORKTREES_ROOT` | set locally | task worktrees |
| `TASK_WORKTREE_ROOT` | set locally | temporary task worktrees |
